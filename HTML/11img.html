Absolute paths always start with the root directory and provide the full path to the file or directory. For example, the absolute path to the home directory of a user named "waqas" would be "/home/waqas". On the other hand, a relative path is a path to a file or directory that is relative to the current directory.

Absolute Path Example

file:///C:/WD/WD_02_April_10AM/HTML/tag_vs_ele.png

OR

C:\WD\WD_02_April_10AM\HTML\assets\images\imageTagInHTML.jpg

Relative Path Example

tag_vs_ele.png

assets/images/imageTagInHTML.jpg
